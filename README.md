# Rawal DSS — Decision Support System for Reservoir Sustainability

A web-based decision support system for the Rawal Dam reservoir, built as a Final Year Design Project at NUST Institute of Civil Engineering (NICE).

## Overview
Combines 25 years of Landsat satellite imagery, SRTM terrain data, and PMD climate records with a calibrated CA-Markov forecasting engine to:
- Quantify historical watershed urbanization
- Forecast reservoir end-of-life
- Simulate policy interventions (check dams, zoning, afforestation)
- Rank scenarios via multi-criteria decision analysis

## Technology
- Python pipeline (rasterio, geopandas, numpy, scipy) for geospatial analysis
- Vanilla JS + Plotly + Leaflet for the web interface
- Hosted free on GitHub Pages
